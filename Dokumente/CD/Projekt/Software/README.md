## DA Feldkemper

### Prototyp öffnen

Für die Ansicht des Prototypen wird ein Browser benötigt.

Der Prototypen kann durch Öffnen von problem.html im Ordner Projekt/Software/Assistenzsystem angesehen werden.

### Prototyp bearbeiten

Für die Veränderung des Prototypen wird Axure (mindestens) Version 8 benötigt.
Die Datei findest sich unter Projekt/Software/Prototyp.rp.
